This is a sklearn based classification framework that given a input data set performs either a binary or multi class classification tasks. 
The framework provides the following functionalities :-

1. Missing Data Imputation
2. Outlier Detection/Removal (Multiple Options offered)
3. Feature Selection (Multiple Options offered)
4. Hyperparameter Selection (Both Random and Grid)
5. Classification Training (Multiple versions of Both Parametric and Non Parametric offered)
6. ROC Curve evaluation and optimum operating point calulation
7. PR/ROC Curves and final Performance Statistics on Test Set

Please run/look at Classification_RunBook_BreastImplantDS.ipynb for a live example and more details on the Breast Implant dataset from sklearn
